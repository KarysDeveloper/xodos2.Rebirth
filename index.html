What is XoDos:Re?

The evolution of the original XoDos project, rebuilt from the ground up

XoDos:Re is a complete Linux desktop environment that runs natively on Android devices. Inspired by the tiny_computer project, XoDos:Re brings the Linux experience to your pocket with no root and no complicated setup required.

Unlike other solutions, XoDos:Re is designed to not interfere with Termux, Termux X11 or other Android apps. It's a pure standalone experience: download, install, and you'll have a full Linux desktop with Wine, GPU drivers, and hundreds of applications available.

Built with Flutter/Dart, XoDos:Re integrates a terminal emulator (xterm), a VNC client (AVNC), and a native X11 client — all in a single app that gives you access to a world of possibilities.

How It Works

The hybrid architecture that makes Linux on Android possible

01

PRoot — The Linux Engine

XoDos:Re uses PRoot to create a Linux (Debian) environment inside Android without needing root. PRoot intercepts system calls and translates them, creating a complete and functional Linux user space.

02

Hybrid Desktop — Native + PRoot

Since v1.0.6, XoDos:Re offers a native Xfce4 desktop running alongside the PRoot environment. Native ARM64 (Bionic) apps and PRoot Linux apps coexist on the same desktop — the best of both worlds.

03

Display: VNC, X11 & Terminal

Three display modes: AVNC (built-in VNC client), Termux X11 (native X11 with better performance), and Terminal (xterm for direct access). Switch between them depending on your needs.

04

GPU Drivers & Wine

Graphics drivers like VirGL, ANGLE, Turnip, GL4ES and Venus VirtIO enable hardware acceleration. Combined with Wine + box64 + DXVK, you can run Windows apps and games with DirectX-to-Vulkan translation.

05

App Store & Quick Commands

Since v1.0.7, a native App Store lets you install Blender, GIMP, Firefox, VSCode and more with one click. Quick Commands make it easy to install additional apps like WPS Office, Telegram, Discord Lite, etc.
